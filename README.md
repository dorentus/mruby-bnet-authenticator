#mruby-bnet-authenticator
Battle.net Mobile Authenticator (mruby class)

## install by mrbgems
- add conf.gem line to `build_config.rb`

```ruby
MRuby::Build.new do |conf|

    # ... (snip) ...

    conf.gem :git => 'https://github.com/dorentus/mruby-bnet-authenticator.git'
end
```
